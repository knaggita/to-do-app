Introduction

Project Management App
It enables users keep track of their tasks and contacts.

The purpose of the Task Manager template is to assist in remembering tasks that need to be done and for one to keep track of their contacts. This could be used by anyone who needs to keep track of their tasks (one-time or recurring) and avoid forgetting them in a busy daily life. Functions

This web app performs the following functions:

  • Allows entering tasks
  • Allows entering contacts
  • Allows the admin to add more users
  • Shows all contacts
  • Shows the action plan

The user should login to add contacts and tasks.
